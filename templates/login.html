CogniAccess

Cognitive Accessibility Research Project

I built this tool to help students with dyslexia, ADHD, and reading difficulties access complex educational text — without needing to ask for help every time.

RandomForest ML Model Predicts font size & line spacing from behaviour data
WordNet NLP Integration Auto-simplifies hard words using Princeton lexical database
Real-time Behaviour Tracking Logs hover patterns to improve ML recommendations
Full Accessibility Suite 5 fonts · Voice input · Text-to-speech · High contrast
Python Flask NLTK scikit-learn SQLite Chart.js Web Speech API
